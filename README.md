# Lovelotte::Rails

		gem wrapper for Lovelotte JQuery plugins http://js.lovelotte.net/jQ/index.php


## Installation

Add this line to your application's Gemfile:

    gem 'lovelotte-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lovelotte-rails
